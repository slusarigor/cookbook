# Cookbook

## Getting Started

Before you clone this repository, be sure you have install the following:
* Postgres -v ~10.3

## Local Development

Clone the latest code from github

`git clone git@github.com:slusarigor/cookbook.git`

Change to the source directory

`cd cookbook`

Run setup script:

```
bin/setup
rails s
```
